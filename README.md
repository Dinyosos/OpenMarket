# OpenMarket

> 去中心化内容营销网络，在 Bittensor 上，让所有项目实现跨平台、批量、低成本的大规模曝光和转化。

## 项目简介

OpenMarket 是一个基于 Bittensor 的去中心化内容营销层，旨在帮助所有有推广需求的项目实现跨平台、批量、低成本的大规模曝光和转化。

## 核心功能

- **跨平台统一分发**：覆盖 Twitter/X、YouTube、小红书、TikTok、知乎、Reddit 等主流平台
- **分设备/分IP 行为模拟**：绕过平台反垃圾检测，模拟真实用户行为
- **AI 全链路内容生成**：自动生成文案、图片、视频等营销内容
- **结果导向评估**：按真实浏览量、互动量、分享量、转化量结算

## 节点架构

- **内容生成分发节点 (Miner)**：负责内容生成与跨平台分发
- **验证节点 (Validator)**：负责评估营销效果和防止作弊
- **Yuma Consensus**：去中心化权重确定与 Token 排放

## 激励机制

- **基准击败模型**：表现最好的节点获得最多 Token 排放
- **独立 Token**：80% 分配给节点，20% 用于生态基金和燃烧

## 评分维度

| 维度 | 权重 |
|------|------|
| 浏览量 | 20% |
| 互动量 | 25% |
| 分享量 | 15% |
| 转化量 | 30% |
| 账号质量 | 10% |
